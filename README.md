### Adding & Removing an article from the saved list
When the user clicks on the bookMark icon within any article's card, it will be added, and if clicked again it will be removed, this can be done in the FeedPage, ArticleDetailsPage, ExplorePage, and SavedPage (anywhere a bookmark is present), and the user will get messages notfying them of what happend.

otherwise they can simply return to the ArticleDetailsPage.
also the textfields will dynamically grow, when a new line is present, and the max height is different for each text field.
